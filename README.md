# Shop Project
A PHP-based customer and order management system.

## Functionality
- View customer list.
- Expand rows to see order history (using `?with-orders=full`).
- Professional gradient UI with glassmorphism.

## Structure
- `src/controllers`: Application logic.
- `src/models`: Database interactions.
- `src/views`: HTML templates.

## Setup
1. Clone the repo.
2. Run `composer install`.
3. Create `.env` from `.env.example`.
4. Import `database.sql`.
